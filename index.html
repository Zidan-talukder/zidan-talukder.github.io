Capstone Project

sEMG Signal-based Prosthetic Hand Control using Embedded ML (TinyML) with Deep Neural Network

Dept. of EEE, UIU | Spring 2023 | Arduino Nano 33 BLE · Edge Impulse · C++
🎖️ Best Capstone Award
  • Developed the complete sEMG signal preprocessing pipeline (filtering, rectification, smoothing, windowing) to produce robust inputs for gesture recognition in a bio-robotic hand control system.
  • Implemented a full Edge Impulse TinyML workflow with FFT spectral analysis. Trained a DNN to classify 4 gestures (97.2% accuracy) and deployed the optimized C++ model to Arduino Nano 33 BLE — achieving 87.6% real-time accuracy.
